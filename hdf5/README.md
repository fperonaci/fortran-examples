  ## Command line tools
  
  * `h5stat`
  * `h5copy` e.g. `h5copy -i input -o output -s /g1u -d /g1u -p`
  * `h5diff` e.g. `h5diff -r input output /green/les /green/les`
  * `h5ls` e.g. `h5ls -r`
  * `h5dump`
